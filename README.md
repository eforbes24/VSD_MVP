# VSD_MVP
Code and documentation for 'Analyzing minimum viable populations in deterministic community models using viability space decomposition'

The main analysis file is the Mathematica notebook 'VSD_MVP'. The data diles (vdat) are all generated by the notebook itself, but those files used in the manuscript are included in this repository. 

Using the Mathematica notebook requires the Dynamica package created by Dr. Randall Beer. Dynamica can be found at his personal website: https://rdbeer.pages.iu.edu/
